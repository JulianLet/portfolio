<!-- summary -->
This project was first developed in one weekend as part of the _GameJam Plus 2024_ competition. After being selected for the acceleration phase we spent more time adding content and polishing the game. In **Tea Party** the players explore the forest and solve puzzles to gather all their friends and complete the game.

<!-- links -->
This game was our submission to the <a href="https://itch.io/jam/gamejamplus-2425" target="_blank" rel="noopener noreferrer">GameJamPlus 24/25</a> where we were one of the 60 teams globally that were selected for the final.

Try this game on <a href="https://oakiki.itch.io/tea-party" target="_blank" rel="noopener noreferrer">itch.io</a>

<!-- my focus -->
In this project my focus was on implementing the different **puzzles** that the player had to complete. To add more verticality into the game I also developed a **navigation system** which included path swapping to navigate between areas.

<!-- reflection -->
This was my first major project after Wind Riders in Unity. During the third semester, we mostly focused on board games in class to work on Game Systems. I noticed a big improvement in my skills, particularly in project setup and organization. After the initial weekend jam, we spent additional time polishing the game and adding content, which revealed small issues that needed to be addressed before continuing. I really enjoyed the weekend crunch to complete the initial submission and the iterative process that followed. Watching the live streams to see if we advanced to the next round of the competition was especially exciting. Overall, this project strengthened my ability to manage a game project from quick prototyping to polished final product while balancing design, technical systems, and gameplay.
